# **App Name**: Connectify Hub

## Core Features:

- Landing Page: A single, responsive landing page to introduce the community platform.
- Secure Authentication: Firebase Authentication integration with email/password and Google OAuth.
- User Profile Creation: Automatic creation of Firestore user documents upon successful authentication.
- Profile Management: Users can view and edit their profiles, including name and profile photo.
- Profile Visibility: Users can toggle their profile visibility between public and private.
- Secure Data Access: Firestore security rules to ensure users can only access their data and public profiles.
- User Dashboard: Central dashboard accessible after login, displaying user information and navigation placeholders.

## Style Guidelines:

- Primary color: Deep Indigo (#3F51B5) to convey trust and stability in the community.
- Background color: Very light Indigo (#E8EAF6) to provide a calm and inviting backdrop.
- Accent color: Soft Violet (#9575CD) to highlight key actions and elements, complementing the indigo tones.
- Body and headline font: 'Inter' for a modern and neutral aesthetic. The body should be legible and the headlines impactful.
- Simple and consistent icons throughout the platform for intuitive navigation.
- Clean and stable layout, focusing on functional components with placeholders for future discussions and groups.
- Minimal animations for smooth transitions and subtle feedback on user interactions.
