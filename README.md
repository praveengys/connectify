# AITSP Community Platform

This is a modern community platform built with Next.js, React, Tailwind CSS, ShadCN UI, and Firebase. It's designed to foster connection, collaboration, and growth among members of the AITSP community.

## Getting Started

To get the development server running:

1.  **Install Dependencies:**
    ```bash
    npm install
    ```

2.  **Environment Variables:**
    Create a `.env` file in the root of the project and add your Firebase and other service configurations.

3.  **Run the Development Server:**
    ```bash
    npm run dev
    ```

Open [http://localhost:9002](http://localhost:9002) with your browser to see the application.

## Core Features

*   **Authentication**: Secure sign-up and login with email/password and Google.
*   **User Profiles**: Customizable member profiles with bio, interests, skills, and more.
*   **Community Forum**: Create discussion threads, post replies, and engage in conversations.
*   **Real-time Chat**: Public and private group chats with live messaging.
*   **Activity Feed**: A central feed to see posts and updates from the community.
*   **Admin & Moderation**: Dashboards for managing users, groups, and content.
*   **AI Assistant**: An integrated AI helper to find information within the community.
